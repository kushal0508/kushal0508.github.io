# Kushal R Portfolio

A modern portfolio website built with Next.js, React, and TypeScript.

## Features

- **Responsive Design**: Fully responsive design that works on all devices
- **Modern Animations**: Smooth animations using Framer Motion
- **Contact Form**: Functional contact form that sends emails via SMTP
- **Project Showcase**: Display of projects with detailed information
- **Experience Timeline**: Timeline of work experience
- **Skills Section**: Display of technical skills with progress bars
- **SEO Optimized**: Optimized for search engines

## Technologies Used

- **Next.js**: React framework for server-side rendering
- **React**: JavaScript library for building user interfaces
- **TypeScript**: Typed superset of JavaScript
- **Framer Motion**: Animation library
- **Tailwind CSS**: Utility-first CSS framework
- **Three.js**: 3D graphics library for background effects
- **Lenis**: Smooth scroll library

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure email settings in `.env.local`:
   ```
   EMAIL_HOST=smtp.gmail.com
   EMAIL_PORT=587
   EMAIL_SECURE=false
   EMAIL_USER=kushalshetty0508@gmail.com
   EMAIL_PASS=your-app-password
   EMAIL_FROM=kushalshetty0508@gmail.com
   EMAIL_TO=kushalshetty0508@gmail.com
   ```
4. Download resume: [Kushal R Resume](resume.txt)
5. Run the development server:
   ```bash
   npm run dev
   ```

## Development

- **Development Server**: `npm run dev`
- **Build for Production**: `npm run build`
- **Start Production Server**: `npm start`
- **Linting**: `npm run lint`
